# Adelic Physics Research Programme: Execution & Deliverables Plan

**Goal:** Investigate whether the adelic (real together with *p*‑adic) completion of the rationals forces dimensionless physical constants — especially the fine‑structure constant — to specific number‑theoretic values, and extract insights about prime numbers via this richer calculus.

**Method:** Build a computational library for *p*‑adic and adelic arithmetic, then construct models (harmonic oscillators, string amplitudes, renormalisation group, spectral action) that impose adelic product constraints. Numerically solve for couplings and mass ratios, compare with experiment, and search for closed‑form expressions.

**Deliverable format per module:**
- A **Python script** (`.py`) containing the core functions/classes, placed in `src/`.
- A **markdown report** (`module_XX_report.md`) that includes narrative, key code snippets (for illustration), printed outputs (tables, numbers), and embedded plots (`![plot](images/plot_XX.png)`).
- **Image files** (`.png`) saved in an `images/` directory.
- Any **data files** (JSON, CSV) in a `data/` directory.
- All outputs are integrated into the final GitHub repository with a top‑level `README.md` and a GitHub Pages configuration so the markdown reports render publicly.

**Execution workflow:**  
For each module, the LLM:
1. Writes the Python code (script) and requests the UI to execute it.
2. Receives the output (print statements, errors) and plots generated.
3. Debugs iteratively by modifying the code and re‑executing.
4. Assembles the final markdown report, embedding the outputs and linking the images.
5. Saves the clean script, report, images, and data into the repository.

## Module 1: Foundational Library — *p*‑adic and Adelic Arithmetic

**Objective:** Implement core number‑theoretic objects: `PadicNumber` and `Adele`, verify the product formula and Ostrowski’s theorem numerically.

**Python script:** `src/padic.py` and `src/adele.py`
- `PadicNumber(prime, rational)`: stores valuation, unit, norm. Implements arithmetic via rational operations, `norm()` returns $p^{-v}$. Hensel expansion method.
- `Adele(rational)`: given a `Fraction`, creates adele with real component as `float` (or high precision) and `PadicNumber` for a finite set of primes (up to a bound, or only primes dividing the rational). Method `norm_product()` returns exact product of all local norms as `Fraction(1,1)`.
- Unit tests (doctest or assert blocks) included in the script; LLM runs them and captures output.

**Markdown report:** `module_01_report.md`
- Introduction to Ostrowski’s theorem and adeles.
- Table of example rationals with real norm, *p*‑adic norms for p=2,3,5,7,11, total product.
- Verification that the product equals 1 exactly.
- Bar chart: norms for a selected rational, saved as `images/01_norms_bar.png`.
- Code snippets for the classes.

**Images:** `images/01_norms_bar.png`

**Deliverables:**
- `src/padic.py`
- `src/adele.py`
- `module_01_report.md`
- `images/01_norms_bar.png`
