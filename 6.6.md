## 1. Section 11.1 — Highest-Priority Investigation

### Item 1: Derive $\log(m_\mu/m_e) = 16/3$ from the adelic structure

| Field | Detail |
|:------|:-------|
| Status | ✅ **ADDRESSED — FAILED** |
| Phase 3 deliverable | Thrust F (6.2.py, 6.2.md) |
| Result | 10 independent derivation approaches tested. All failed. The number $16/3 = \frac{2}{3}\sum_f Q_f^2$ comes from SM anomaly cancellation ($\sum_f Q_f^2 = 8$), not from the zeta function. $\Gamma_\infty(1/3)^3 = 15.90 \neq \exp(16/3)$. Worldsheet instanton interpretation fails ($\Delta q = 8/\pi \notin \mathbb{Z}$). No Veneziano cross-ratio equals $16/3$. |
| Classification | **[CODE-EXECUTED]** — exhaustive computational search |

### Item 2: Predict $\log(m_\tau/m_\mu)$

| Field | Detail |
|:------|:-------|
| Status | ✅ **ADDRESSED — FALSIFIED** |
| Phase 3 deliverable | Thrust G (6.3.py, 6.3.md) |
| Result | $\log(m_\tau/m_\mu) = 2.8224$ matches no adelic/zeta expression at $<1\%$ precision. ~20,000 patterns tested (rational fractions, zeta ratios, Gamma_inf combinations, Veneziano CRs). Five symmetry hypotheses (equal log spacing, geometric progression, power law, golden ratio, Koide) all failed. Bonferroni-corrected $p > 0.05$. The coincidence is specific to $\mu/e$ — it does NOT generalize. |
| Classification | **[CODE-EXECUTED]** — systematic search |

### Item 3: Extend to quark masses and CKM mixing angles

| Field | Detail |

## 3. Section 11.3 — Conceptual Open Questions

### Question 1: Is the adelic framework predictive or tautological?

| Field | Detail |
|:------|:-------|
| Status | ✅ **ADDRESSED — ANSWERED** |
| Phase 3 deliverable | Thrust F + G + J (6.2, 6.3, 6.5) |
| Answer | **Neither purely predictive nor purely tautological.** The adelic framework is a **structural meta-theory**: it constrains the *form* of physical laws (functional equations, RG boundedness, rational invariants) without determining specific coefficient values. The identity $\Gamma_\infty = \zeta(1-x)/\zeta(x)$ is mathematically tautological (it IS the zeta functional equation). The structural constraints (Landau pole cancellation, RG boundedness) are non-tautological physical consequences. The search for specific numerical predictions (mass ratios) failed — the coincidence was falsified. |
| Classification | **[LLM-INFERRED]** — synthesis of code-executed results |

### Question 2: What determines the Standard Model gauge group?

| Field | Detail |
|:------|:-------|
| Status | ❌ **NOT ADDRESSED — OPEN** |
| Phase 3 plan | Not assigned |
| Reason | Determining the SM gauge group $SU(3) \times SU(2) \times U(1)$ from the adelic/zeta structure would require: (1) identifying a specific Shimura variety whose CM periods match particle masses, (2) showing that anomaly cancellation selects this specific gauge group, or (3) deriving the gauge group from constraints on the adelic product formula. None of these could be attempted without the blocked computations (11.2.1, 11.2.2). |
| Partial insight | The number of generations (3) appears in both the SM ($N_g = 3$) and the Veneziano symmetry ($a+b+c = 1$, three channels). But this connection is suggestive, not derived. The SO(10) spinor dimension 16 appears in $b_0 = 16/(3\pi)$, but this is coincidental — the 16 in $b_0$ comes from $2 \cdot N_g \cdot \sum Q_f^2$, not from SO(10). |
| Classification | **OPEN** — requires blocked computations |

### Question 3: Are cross-ratios of Veneziano pole positions observable?

| Field | Detail |
|:------|:-------|
| Status | ✅ **ADDRESSED — ANSWERED** |
| Phase 3 deliverable | Thrust F (6.2.py, Approach 7) + Thrust G (6.3.py) |
| Answer | **Empirically, no.** A systematic search for Veneziano pole cross-ratios matching lepton mass ratios found no matches. The pole indices $n_i$ are integers giving rational cross-ratios — but none of these rational numbers match $m_\mu/m_e$, $m_\tau/m_\mu$, or $m_\tau/m_e$ at admissible precision. The rational cross-ratios are mathematically well-defined but do not appear to correspond to observable mass ratios through the compactification dictionary as currently understood. |
| Classification | **[CODE-EXECUTED]** — systematic search through CR space |
